# Pivot

## Syntax
`piv -operation addtional_info data`.

`data` -> data in requested type.
### Operations
- -s shift from one type to another
- -e encode raw data to a type
- -d decode type to raw data
- -h prints help screen

### Types ( Addtional_info )
[ ] `b` . binary

[ ] `o` . octa

[ ] `h` . hexadecimal

[ ] `d` . decimal

[ ] `raw` . normal text

[ ] `b64` . base64
